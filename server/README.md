# Server (Day 1 setup)

This is the backend starter setup using Express + TypeScript.

## What is ready
- TypeScript config
- Express app
- `GET /` route
- `GET /api/health` route
- `.env.example`

## Run locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start dev server:
   ```bash
   npm run dev
   ```
3. Test in browser or Postman:
   - `http://localhost:5000/`
   - `http://localhost:5000/api/health`

## Notes
If `npm install` fails due to network/policy, run it from your local machine where npm access is allowed.
